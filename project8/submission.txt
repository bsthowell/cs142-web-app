Story number and name: 3 - Visibility control on photos
Story points: 4
Brief description to see the story in action: 
Select a file to upload by pressing the button in the toolbar. Either select a sharing list, or allow anyone to view. Log into different accounts to see that only the original user and those selected can view the photo.

Story number and name: 5 - Deleting Comments, Photos, and Users
Story points: 4
Brief description to see the story in action: 
There are delete icons next to photos and comments belonging to the logged in user. When clicked, these items are deleted. The logged in user also has a delete icon displayed on their own user detail page which deletes their entire account when clicked.

Story number and name: 6 - Photo like votes
Story points: 4
Brief description to see the story in action:
Each photo has a button that toggles between like and unlike for any photo on the web app. When the logged in user likes a photo, a small thumbs up icon appears next to that photo. Likes and unlikes are reflected immediately in the like count beneath the photo too.

Total Story Points: 12
